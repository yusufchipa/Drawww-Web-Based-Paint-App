## ✨ About

**Drawww** is a feature-rich browser drawing application built using **React**, **Vite**, and the **HTML5 Canvas API**.

It combines powerful drawing functionality with a fun, colorful interface inspired by modern design trends like glassmorphism, gradients, micro-animations, and playful visuals.

Think of it as:

> 🎨 MS Paint + ✨ modern web magic + 🌈 Gen Z aesthetics

## 🖌️ Drawing Features

### ✏️ Pencil

Precise, lightweight strokes for detailed sketching.

### 🖌️ Brush

Smooth natural drawing with balanced opacity.

### 🖍️ Marker

Semi-transparent highlighter effect for layering.

### 🧽 Eraser

Clean canvas matching with accurate color replacement.

### 🪣 Fill Tool

Flood-fill enclosed regions instantly.

### 🔤 Text Tool

Place editable text directly onto the canvas.

## ⚡ Getting Started

### Clone Repository

```bash
git clone https://github.com/yusufchipa/Drawww-Web-Based-Paint-App.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```
